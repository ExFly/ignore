# Ignore
Download .gitignore files from the GitHub [gitignore](https://github.com/github/gitignore) repository!

### Using it
Get the binary:
```
$ go install github.com/jonatasbaldin/ignore
```

Listing files:
```
$ ignore -list
Actionscript
Ada
Agda
Android
AppEngine
...
```

Downloading a file:
```
$ ignore Python
```
